# 🤖 Telegram Bot Project

Bu layihə Telegram Bot API istifadə edilərək hazırlanmış sadə avtomatlaşdırılmış mesaj botudur. Layihənin əsas məqsədi CI/CD proseslərini, avtomatik workflow məntiqini və GitHub Actions istifadəsini praktik şəkildə öyrənməkdir.

Bot müəyyən vaxt intervallarında və ya workflow işə düşdükdə Telegram üzərindən motivasiyaedici mesaj göndərir:
> **“Həyat gözəldir”**

## 🚀 Layihənin Məqsədi
Bu layihə aşağıdakı mövzuları öyrənmək və praktiki şəkildə tətbiq etmək üçün hazırlanmışdır:
* 📲 Telegram bot inteqrasiyası
* 🐍 Python scripting
* 🐙 GitHub Actions istifadəsi
* 🔄 CI/CD workflow məntiqi
* ⚙️ Automation prosesləri
* 🔐 Environment variables və secrets istifadəsi

## ⚙️ İstifadə olunan texnologiyalar
* **Python**
* **Telegram Bot API**
* **GitHub Actions**
* **Requests library**

## 🔄 Əsas funksionallıqlar
* Telegram üzərindən avtomatik mesaj göndərilməsi
* Müəyyən vaxt intervallarında (cron job) workflow işləməsi
* GitHub push zamanı avtomatik execution
* Təhlükəsiz token istifadəsi üçün secrets management

## 🧪 CI/CD Prosesi
Layihədə GitHub Actions istifadə edilərək aşağıdakı proseslər avtomatik icra olunur:
1. Dependency-lərin yüklənməsi (`pip install`)
2. Python scriptinin işə salınması
3. Telegram mesajının göndərilməsi
4. Workflow-un schedule (vaxt nizamlaması) və ya push event ilə avtomatik işləməsi

## 📌 Nəticə
Bu layihə başlanğıc səviyyədə CI/CD və avtomatlaşdırma məntiqini öyrənmək üçün hazırlanmış sadə və praktik Telegram bot nümunəsidir.
